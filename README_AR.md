# نظام Al-swadi Pro - تطبيق الهاتف المحمول

تطبيق Flutter متكامل لإدارة البيانات المالية اليومية لعيادة وصيدلية.

## الميزات

- **إدخال البيانات اليومية**: 6 حقول لإدخال البيانات المالية (دخل العيادة، دخل الصيدلية، سداد الشركات، مشتريات نقدية، المصروفات)
- **حسابات تلقائية**: حساب الإجمالي النهائي تلقائياً
- **عرض المعاملات**: عرض جميع المعاملات المضافة في قائمة
- **القفل التلقائي**: قفل النظام بكلمة المرور (Alswadi773500)
- **واجهة عربية كاملة**: جميع النصوص بالعربية مع أرقام إنجليزية
- **تصميم استجابي**: يعمل على جميع أحجام الشاشات

## المتطلبات

- Flutter SDK 3.10.4 أو أحدث
- Android SDK (لبناء APK)
- Xcode (لبناء iOS)

## التثبيت

1. استنساخ المشروع:
```bash
cd /home/ubuntu/al_swadi_mobile
```

2. تثبيت المكتبات:
```bash
export PATH="/home/ubuntu/flutter/bin:$PATH"
flutter pub get
```

## التشغيل

### على محاكي Android:
```bash
flutter run
```

### على جهاز فعلي:
```bash
flutter run -d <device_id>
```

## البناء

### بناء APK (Android):
```bash
flutter build apk --release
```

الملف سيكون في: `build/app/outputs/flutter-apk/app-release.apk`

### بناء IPA (iOS):
```bash
flutter build ios --release
```

كلمة المرور الافتراضية للقفل: `Alswadi773500`
